# OtoWash · Strategic Investment Report

## 📋 Overview

OtoWash is a professional, interactive investment report dashboard designed in the style of top-tier consulting firms (Big 4). It presents a complete business case for deploying an automated "Touchless 360°" car wash infrastructure in the rapidly growing Dakar automotive market.

### Key Features

- **📊 Executive Dashboard** — Key investment metrics at a glance
- **📈 Market Analysis** — Interactive charts showing market share and trends
- **🎯 SWOT Matrix** — Strategic positioning analysis
- **💰 Pricing Strategy** — Tiered service packages with recommendations
- **🏗️ Business Model Canvas** — Complete business architecture
- **🚢 Supply Chain Strategy** — China to Dakar import roadmap
- **📉 Financial Projections** — CAPEX, EBITDA, revenue forecasts with Chart.js
- **🗺️ Deployment Roadmap** — 7-step strategic timeline
- **💡 Strategic Recommendations** — Key investment insights
- **📱 Fully Responsive** — Optimized for desktop, tablet, and mobile
